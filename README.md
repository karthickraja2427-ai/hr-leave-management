HR Leave Management Tool

Overview

The HR Leave Management Tool is a web application designed to simplify the process of managing employee leave requests. It allows employees to apply for leave online while enabling HR or administrators to review, approve, or reject requests efficiently.

Features

- Employee login
- Apply for leave
- View leave history
- Approve or reject leave requests (Admin/HR)
- Leave status tracking
- Dashboard for employees and administrators
- User-friendly interface

Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Version Control: Git & GitHub

Installation

1. Clone this repository.
2. Move the project folder to your local web server (XAMPP/WAMP).
3. Import the SQL database into MySQL.
4. Update the database configuration file if required.
5. Start Apache and MySQL.
6. Open the project in your browser.

Usage

- Login using your account.
- Employees can submit leave requests.
- HR/Admin can review and manage leave requests.
- Users can track the status of their leave applications.
